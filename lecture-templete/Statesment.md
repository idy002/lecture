### Problem A

- 输入文件：prob.in
- 输出文件：prob.out
- 时间限制：1 second
- 空间限制：128 MB

#### 题目描述

小明想求a + b等于多少。

#### 输入格式

第一行包含两个数$a \; b$。

#### 输出格式

第一行包含一个数：$c$，表示结果。

#### 样例

| 样例输入 | 样例输出 |
| :--- | ---- |
| 3 4  | 7    |



#### 数据范围

- 对于$10\%$的数据，$1 \leq n \leq 100$；

- 对于$100\%$的数据，$1 \leq n \leq 10^5$。
